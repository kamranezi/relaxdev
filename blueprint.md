# RuVercel PaaS - Blueprint

## 1. Обзор проекта

**Цель:** Создать веб-интерфейс для PaaS-платформы "RuVercel", аналога Vercel, работающей на Yandex Cloud.

**Стек:**
- **Фронтенд:** Next.js (App Router)
- **Стилизация:** Tailwind CSS
- **Бэкенд (предполагаемый):** NestJS
- **Инфраструктура (управляемая пользователем):** Yandex Cloud (Serverless Containers, Managed Redis, VM)
